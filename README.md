# Interfaces-Figma-PROA
Este repositório reúne as atividades de Interface e Prototipagem desenvolvidas no Figma, propostas pelo PROA e orientadas pela professora Débora, responsável pela Equipe 1 de Web Java.

Aqui estão organizados os links de protótipos com layouts, wireframes e estudos de UI/UX realizados ao longo do curso.

O objetivo é documentar minha evolução em design de interfaces, aplicando conceitos de usabilidade, acessibilidade, hierarquia visual, tipografia e paleta de cores.

Os projetos foram desenvolvidos com foco em boas práticas de experiência do usuário (UX) e design de interface (UI), seguindo as orientações e os desafios propostos em aula.
